Usage rules

Hierarchy

One primary per view. Pair primary with outlined or ghost for secondary actions — never two primaries side-by-side.

Size

XL in hero sections. LG inside cards. MD/SM for dense UI (filters, table rows).

Radius

Buttons are full pills (50px). Do not square them off — it breaks brand voice and reads as a generic admin UI.
